# 👤 Interface de Usuário

Uma interface de usuário (UI) avançada e altamente funcional desenvolvida para permitir a gestão completa de perfis profissionais. O projeto foca em **usabilidade**, oferecendo uma experiência fluida para edição de dados pessoais, gestão de fuso horário e upload de ativos visuais.

## 🚀 Sobre o Projeto

Esta aplicação foi construída para simular um painel de configurações de conta (Account Settings), onde o utilizador tem controle total sobre sua presença digital. A interface prioriza o feedback visual imediato e a facilidade de navegação.

### Funcionalidades Principais:
*   **Edição de Perfil:** Alteração dinâmica de nome, país e área de atuação.
*   **Gestão de Fuso Horário:** Seleção otimizada para garantir a precisão em agendas globais.
*   **Preview de Foto de Perfil:** Sistema de upload com visualização instantânea da imagem na própria página antes da confirmação.
*   **Gestão de Portfólio:** Módulo de upload de ficheiros para exibição de projetos e documentos.
*   **Interface Responsiva:** Layout adaptável que mantém a hierarquia visual em dispositivos móveis e desktop.

## 🛠️ Tecnologias Utilizadas

*   **React (Hooks & Componentes Funcionais):** Para uma interface reativa e performática.
*   **Tailwind CSS:** Estilização moderna e utilitária, garantindo um design limpo e responsivo.
*   **TypeScript:** Tipagem estática para maior segurança no desenvolvimento e manutenção do código.
*   **Mobile-First Design:** Foco inicial na usabilidade em ecrãs pequenos.

## ⚙️ Diferenciais de Usabilidade

1.  **Feedback Instantâneo:** Validação de campos e previews em tempo real para evitar erros do utilizador.
2.  **Clean UI:** Aplicação de espaçamentos e tipografia que reduzem a carga cognitiva.
3.  **Acessibilidade:** Elementos de formulário estruturados para navegação via teclado e leitura clara.
